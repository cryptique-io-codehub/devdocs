## Installation Methods

Cryptique can be installed in two ways:

| Method | Best For | Auto-Init |
|--------|----------|-----------|
| **NPM Package** | React, Next.js, Vue, modern JS apps | Manual |
| **CDN Script** | Static sites, quick testing, no build step | Automatic |

---

## NPM Package

### Install

<CodeGroup>
```bash npm
npm install cryptique-sdk
```

```bash yarn
yarn add cryptique-sdk
```

```bash pnpm
pnpm add cryptique-sdk
```
</CodeGroup>

### Import & Initialize

```javascript
import Cryptique from 'cryptique-sdk';

Cryptique.init({
  siteId: 'your-site-id',
  autoEvents: true
});
```

## CDN Script Tag

For static sites or quick testing, use the CDN version. The SDK auto-initializes using attributes on the script tag.

### Basic Installation

```html
<script 
  src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
  site-id="your-site-id" 
  auto-events="true">
</script>
```

<Info>
  Place the script tag in your `<head>` or at the end of `<body>`. Earlier placement ensures you capture all page interactions.
</Info>

### Script Attributes

| Attribute | Type | Default | Description |
|-----------|------|---------|-------------|
| `site-id` | string | *required* | Your Cryptique Site ID |
| `auto-events` | boolean | `true` | Enable/disable automatic event tracking |
| `auto-events-disabled-events` | string | `""` | Comma-separated list of auto-events to disable |
| `auto-events-disabled-paths` | string | `""` | Comma-separated list of URL paths to exclude |

### CDN Examples

<Tabs>
  <Tab title="Basic">
    ```html
    <script 
      src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
      site-id="your-site-id" 
      auto-events="true">
    </script>
    ```
  </Tab>

  <Tab title="Disable Specific Events">
    ```html
    <script 
      src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
      site-id="your-site-id" 
      auto-events="true"
      auto-events-disabled-events="page_scroll,text_selection">
    </script>
    ```
  </Tab>

  <Tab title="Exclude Paths">
    ```html
    <script 
      src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
      site-id="your-site-id" 
      auto-events="true"
      auto-events-disabled-paths="/admin,/internal">
    </script>
    ```
  </Tab>

  <Tab title="Disable All Auto-Events">
    ```html
    <script 
      src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
      site-id="your-site-id" 
      auto-events="false">
    </script>
    ```
  </Tab>
</Tabs>

### Using with CDN (After Load)

When using the CDN, the `Cryptique` object is available globally after the script loads:

```html
<script 
  src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
  site-id="your-site-id" 
  auto-events="true">
</script>

<script>
  // Track a custom event
  Cryptique.track('Button Clicked', { 
    buttonId: 'cta-main' 
  });
</script>
```

## Verifying Installation

After installation, verify Cryptique is working:

<Steps>
  <Step title="Check Console">
    Open your browser's developer console. You should see no errors related to Cryptique.
  </Step>
  
  <Step title="Check Network Tab">
    In the Network tab, filter by "cryptique". You should see requests to `api.cryptique.io` for event tracking.
  </Step>
  
  <Step title="Check Dashboard">
    Log into [app.cryptique.io](https://app.cryptique.io) and view your **Live Events** feed. You should see `page_view` events appearing in real-time.
  </Step>
</Steps>

## TypeScript Support

The NPM package includes TypeScript definitions out of the box:

```typescript
import Cryptique from 'cryptique-sdk';

interface SwapEventProps {
  fromToken: string;
  toToken: string;
  amount: number;
  slippage: number;
}

Cryptique.track<SwapEventProps>('Swap Completed', {
  fromToken: 'ETH',
  toToken: 'USDC',
  amount: 1.5,
  slippage: 0.5
});
```
