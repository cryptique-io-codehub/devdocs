## Overview

This guide will walk you through:

1. Creating your Cryptique account
2. Installing the SDK
3. Initializing tracking
4. Adding your first smart contract
5. Verifying data flow

<Steps>
  <Step title="Create Your Account">
    Sign up for a free Cryptique account at [app.cryptique.io/signup](https://app.cryptique.io/signup).
    
    After signing up, you'll be taken to your dashboard where you can find your **Site ID**.
    
    <Info>
      Your Site ID is located in **Settings → Sites**. You'll need this to initialize the SDK.
    </Info>
  </Step>

  <Step title="Install the SDK">
    Choose your preferred installation method:

    <Tabs>
      <Tab title="NPM">
        ```bash
        npm install cryptique-sdk
        ```
      </Tab>
      <Tab title="Yarn">
        ```bash
        yarn add cryptique-sdk
        ```
      </Tab>
      <Tab title="CDN">
        ```html
        <script 
          src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
          site-id="your-site-id" 
          auto-events="true">
        </script>
        ```
      </Tab>
    </Tabs>
  </Step>

  <Step title="Initialize Cryptique">
    <Tabs>
      <Tab title="NPM / ES Modules">
        ```javascript
        import Cryptique from 'cryptique-sdk';

        Cryptique.init({
          siteId: 'your-site-id'
        });
        ```
      </Tab>
      <Tab title="React / Next.js">
        ```jsx
        // In your _app.js or layout component
        import { useEffect } from 'react';
        import Cryptique from 'cryptique-sdk';

        function MyApp({ Component, pageProps }) {
          useEffect(() => {
            Cryptique.init({
              siteId: 'your-site-id',
              autoEvents: true
            });
          }, []);

          return <Component {...pageProps} />;
        }

        export default MyApp;
        ```
      </Tab>
      <Tab title="CDN">
        ```html
        <!-- SDK auto-initializes with script attributes -->
        <script 
          src="https://cdn.cryptique.io/scripts/analytics/1.0.1/cryptique.script.min.js" 
          site-id="your-site-id" 
          auto-events="true">
        </script>
        ```
      </Tab>
    </Tabs>

    <Check>
      Once initialized, Cryptique automatically starts tracking page views, clicks, and other events.
    </Check>
  </Step>

  <Step title="Track Your First Custom Event">
    ```javascript
    // Track a swap completion
    Cryptique.track('Swap Completed', {
      fromToken: 'ETH',
      toToken: 'USDC',
      amount: 1.5,
      slippage: 0.5
    });
    ```

    <Tip>
      Custom events appear in your dashboard within seconds. Use them to track Web3-specific actions like swaps, stakes, mints, and more.
    </Tip>
  </Step>

  <Step title="Add Your Smart Contract">
    Navigate to **Settings → Contracts** in your dashboard:

    1. Click **Add Contract**
    2. Enter your contract details:
       - **Contract Address**: Your deployed contract address
       - **Contract Name**: A friendly name for identification
       - **Token Symbol**: The symbol (e.g., USDC, NFT)
       - **Chain**: Select from the dropdown (35+ supported)
       - **Decimals**: Token decimal places (usually 18)
    3. Click **Save**

    <Info>
      Cryptique automatically indexes all historical transactions from your contract's deployment block. This may take a few minutes depending on transaction volume.
    </Info>
  </Step>

  <Step title="Verify Data Flow">
    Open your dApp in a browser and perform some actions:

    1. Navigate between pages
    2. Click some buttons
    3. Connect a wallet (if applicable)

    Then check your Cryptique dashboard—you should see:
    - **Live Events**: Real-time event stream
    - **Sessions**: Active user sessions
    - **Wallets**: Connected wallet addresses (after connection)

    <Check>
      If you see data flowing, congratulations! Cryptique is now tracking your dApp.
    </Check>
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Events not appearing in dashboard">
    - Verify your Site ID is correct
    - Check browser console for errors
    - Ensure the SDK is initialized before tracking events
    - Confirm you're not blocking the SDK with ad blockers
  </Accordion>
  
  <Accordion title="Wallet connections not tracked">
    - Cryptique auto-detects wallet connections via EIP-6963 and `window.ethereum`
    - Ensure the wallet provider is available before Cryptique initializes
    - Check that `wallet_connected` events appear in your session data
  </Accordion>
  
  <Accordion title="Contract transactions not showing">
    - Indexing historical data can take a few minutes
    - Verify the contract address and chain are correct
    - Check that your plan includes sufficient API calls
  </Accordion>
</AccordionGroup>
