## Overview

Cryptique is designed with privacy in mind. We collect only what's necessary for analytics, respect user privacy preferences, and comply with major privacy regulations.

---

## What We Collect

### Off-Chain Data (via SDK)

| Data Type | Description |
|-----------|-------------|
| **Sessions** | Session ID, duration, pages visited |
| **Events** | Clicks, scrolls, form submissions, custom events |
| **UTM Parameters** | Campaign tracking data |
| **Device Info** | Browser, OS, screen size |
| **Geographic Info** | Country, region (derived from IP, not stored) |

### Behavioral Data

| Data Type | Description |
|-----------|-------------|
| **Session recordings** | Mouse movements, clicks, scrolls (privacy-safe) |
| **Heatmaps** | Aggregated click/scroll data |
| **Rage clicks** | Frustration signals |
| **Dead clicks** | Non-responsive click signals |

### On-Chain Data

| Data Type | Description |
|-----------|-------------|
| **Wallet addresses** | Public, pseudonymous |
| **Transaction history** | From public blockchain data |
| **Token holdings** | Readable from blockchain |
| **Smart contract interactions** | Public on-chain events |

---

## What We DON'T Collect

<Warning>
  Cryptique **never** collects, stores, or has access to:
</Warning>

| Data Type | Status |
|-----------|--------|
| Private keys | ❌ Never |
| Seed phrases | ❌ Never |
| Passwords | ❌ Never |
| Authentication credentials | ❌ Never |
| Off-chain messages/communications | ❌ Never |
| Form field values (by default) | ❌ Never (unless you explicitly track them) |
| Anything not public on blockchain | ❌ Never |

## User Opt-Out

### SDK Method

Users can opt out of tracking:

```javascript
// Opt out of all tracking
Cryptique.optOut();
```

### Do Not Track

Respect browser DNT settings:

```javascript
const dnt = navigator.doNotTrack === '1' || window.doNotTrack === '1';

if (dnt) {
  Cryptique.optOut();
}
```

## GDPR Compliance

Cryptique is fully GDPR compliant:

| Requirement | Implementation |
|-------------|----------------|
| **Lawful basis** | Legitimate interest for analytics |
| **Pseudonymous data** | Wallet addresses are pseudonymous (not PII) |
| **Right to access** | Users can request their data |
| **Right to deletion** | Users can request data deletion |
| **Data portability** | Data export available |
| **Breach notification** | 72-hour notification commitment |

### Data Subject Requests

For GDPR requests (access, deletion, correction):

- Email: [hello@cryptique.io](mailto:hello@cryptique.io)
- Response time: Within 30 days

### Implementing Consent

```javascript
// Wait for consent before initializing
function handleCookieConsent(consentGiven) {
  if (consentGiven) {
    Cryptique.init({
      siteId: 'your-site-id',
      autoEvents: true
    });
  } else {
    // Initialize but opt out
    Cryptique.init({
      siteId: 'your-site-id',
      autoEvents: false
    });
    Cryptique.optOut();
  }
}
```

### Cookie Consent Implementation

```javascript
// Check for existing consent
const consent = localStorage.getItem('cryptique_consent');

if (consent === 'accepted') {
  Cryptique.init({ siteId: 'your-site-id' });
} else if (consent === 'rejected') {
  Cryptique.init({ siteId: 'your-site-id', autoEvents: false });
  Cryptique.optOut();
} else {
  // Show consent banner
  showConsentBanner();
}
```

## Data Deletion

### For Users

Request deletion by emailing [hello@cryptique.io](mailto:hello@cryptique.io) with:
- Your user identifier or wallet address
- The specific data you want deleted

Processing time: Within 30 days

### Via SDK

```javascript
// Delete user profile
Cryptique.people.deleteUser();

// Reset session
Cryptique.reset();
```

<Note>
  On-chain blockchain data cannot be deleted as it exists on public ledgers. We can only delete our indexed copies.
</Note>
